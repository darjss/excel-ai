# Vertical wedge in marketing, horizontal engine

The platform could serve any Excel-to-website conversion, but v1 markets and ships exactly one thing: order spreadsheets turned into Buyer-facing order portals for wholesale/bakery/farm Suppliers. The Extraction engine stays generic (it never knows about "orders"); the wedge lives only in portal template families, landing copy, and outreach targets. Chosen because horizontal marketing competes head-on with Glide's SEO and the AI-builder brands with no differentiation, while the order wedge scored highest on demand evidence, willingness to pay, v1 simplicity (no payments, no Buyer accounts needed), and reachability (2026-07 niche research). Widening later means adding template families and landing pages, not re-architecting.
